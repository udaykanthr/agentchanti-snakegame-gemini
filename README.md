# Cute Snake Game

A modern, polished Snake game built with Python and Pygame. This version features smooth animations, a "cute" pastel aesthetic, and juicy visual feedback like screen shakes and particle effects.

## Features

- **Cute Visuals**: A pastel-colored snake (Pink head, Green body) with expressive eyes.
- **Juicy Feedback**: 
  - **Screen Shake**: The screen shakes when you eat or hit a wall.
  - **Firecrackers**: Particle explosions occur when the snake consumes an apple.
  - **Animations**: Pulsing effects for the apple and the snake's head.
- **Life System**: You start with 3 lives. Hitting a wall or your own tail costs a life and respawns the snake.
- **Modern Palette**: Dark forest green background with a subtle grid for better readability.

## Requirements

- Python 3.7+
- Pygame
